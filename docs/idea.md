# Product Vision

CareerLens is a modern AI-powered Career Operating System.

It helps professionals improve every aspect of their career from a single platform.

CareerLens combines

- Resume Intelligence
- LinkedIn Management
- AI Content Creation
- Career Insights
- Job Discovery

into one seamless workspace.

CareerLens is NOT an AI chatbot.

CareerLens is NOT a generic admin dashboard.

CareerLens is a premium SaaS platform designed for professionals.

# Design Philosophy

The product should look like it was designed by senior product designers.

Never generate AI-looking interfaces.

No glowing gradients.

No futuristic effects.

No colorful dashboards.

No glassmorphism overload.

No Dribbble concepts.

Prioritize

- simplicity
- usability
- whitespace
- hierarchy
- consistency

# AI Rules

AI is only a capability.

It is not the visual identity.

Never use AI-themed illustrations.

Never use robots.

Never use glowing brains.

Never use neural graphics.

---

# Development Rules

Always generate production-ready React components.

Use TypeScript.

Use reusable components.

Separate UI from business logic.

Follow clean architecture.

Never generate placeholder code unless explicitly requested.

Create scalable folder structures.
